# API de Tarefas com Flask

Projeto de estudo para criação de uma API REST utilizando Flask e SQLite.

## Tecnologias

- Python
- Flask
- SQLite
- HTML
- JavaScript

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Editar tarefa
- Deletar tarefa

## Como rodar o projeto

Clone o repositório:

git clone https://github.com/denv-luis/api-tarefas-flask

Entre na pasta:

cd api-tarefas-flask

Crie o ambiente virtual:

python -m venv venv

Ative o ambiente:

source venv/bin/activate

Instale as dependências:

pip install -r requirements.txt

Execute:

python app.py
